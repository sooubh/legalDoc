## Analyzing a Document

### Step 1: Navigate to Upload Page
- Click "Upload" in the sidebar (desktop)
- Or tap "Upload" in the bottom navigation (mobile)

### Step 2: Choose Input Method

#### Option A: Paste Text
1. Click in the text area
2. Paste your document text (Ctrl+V or Cmd+V)
3. Text can be any length

#### Option B: Upload PDF
1. Click "Upload PDF" button
2. Choose a PDF file from your device
3. Or drag and drop a PDF into the upload area
4. App will extract text automatically
5. OCR will activate for scanned PDFs

#### Option C: Try a Sample
1. Click "Try Sample Contract"
2. Select language (EN or HI)
3. Choose from:
   - Service Agreement
   - Mutual NDA
   - Residential Lease
4. Sample will load automatically

### Step 3: Configure Analysis
- **Language**: English or Hindi for analysis results
- **Simplification Level**:
  - **Professional**: For legal professionals, preserves terminology
  - **Simple**: Plain language for general users
  - **ELI5**: Extremely simplified (Explain Like I'm 5)

### Step 4: Submit
- Click "Analyze Document"
- Wait for analysis to complete (10-60 seconds depending on length)
- Larger documents take longer due to chunking

## Using Visualizations

Navigate to "Visuals" page to see auto-generated diagrams.

### Process Flow Diagrams
**Shows**:
- Process steps (e.g., termination, renewal, dispute resolution)
- Decision points (Yes/No branches)

**How to Use**:
- Scroll to view entire diagram
- Click "Fullscreen" for better view
- Diagrams are in Mermaid format

### POV Timelines
**Three perspectives**:
1. **Court Perspective**: Legal process timeline
2. **Receiver Perspective**: Party receiving notice
3. **Overall Timeline**: Combined view

**How to Use**:
- Switch between tabs to see different perspectives
- Each event shows:
  - Title and subtitle
  - Date (when applicable)
  - Description
  - Color-coded by type

### Responsibility Matrix
**Table format showing**:
- Topic/obligation
- Party A responsibilities
- Party B responsibilities
- Related clauses

**Helps You**:
- Compare obligations side-by-side
- Identify imbalances
- Understand who does what

## Chatting About Your Document

Two ways to access chat:

### Method 1: Chat Page
1. Click "Chat" in sidebar
2. Full chat interface appears
3. Type question in input field
4. Press Enter or click Send

### Method 2: Floating Chat Button
1. Click floating chat button (bottom-right on most pages)
2. Chat expands over current page
3. Minimize to continue working

### Using Chat Effectively

#### Quick Questions
Tap any suggested question:
- "What are the key obligations?"
- "What are the main risks?"
- "What are the termination conditions?"
- "What is the notice period?"

#### Custom Questions
Ask anything about the document:
- "Explain clause 5 in simpler terms"
- "What happens if I break this clause?"
- "Can I negotiate this term?"
- "What documents do I need to provide?"

#### Follow-Up Questions
Chat maintains context:
- "Can you explain that further?"
- "What are the exceptions?"
- "Is this common?"

#### Features
- Markdown formatting in responses
- **Bold** and *italic* text
- Bullet points and numbered lists
- Code blocks for examples
- Clickable links

## Managing Your History

### Saving Analyses

#### Option 1: Manual Save
1. After analysis completes
2. Click "Save Analysis" button
3. Confirms save success
4. Appears in history sidebar

#### Option 2: Auto-Save
- Enable in Settings
- Analyses save automatically on completion

### Viewing Saved Analyses

#### Desktop
- History sidebar shows all saved analyses
- Click any item to load
- Sorted by most recent

#### Mobile
- Access from sidebar menu
- Swipe to see options
- Tap to load

### Loading a Previous Analysis
1. Click an item in history
2. Analysis loads instantly
3. Original document also restored
4. All visualizations preserved

### Deleting Analyses
1. Find analysis in history
2. Click delete icon (trash)
3. Confirm deletion
4. Removes from both local and cloud storage

### Storage Details
- **Local**: Last 100 analyses (works offline)
- **Cloud**: Unlimited (requires sign-in)
- **Sync**: Automatic across devices when signed in

## Tips & Best Practices

### For Best Analysis Results

1. **Use Clear Text**
   - Well-formatted documents analyze better
   - Remove excessive whitespace
   - Ensure proper paragraph breaks

2. **Provide Complete Documents**
   - Full contracts better than excerpts
   - Include all parties, dates, signatures
   - Context helps AI understand better

3. **Choose Appropriate Simplification**
   - Professional: If you understand legal terms
   - Simple: For general understanding
   - ELI5: If completely new to legal docs

4. **Try Different Languages**
   - Hindi speakers may understand better in Hindi
   - English provides more technical detail

### For Effective Chat Usage

1. **Be Specific**
   - Instead of "Tell me about this," ask "What are my payment obligations?"
   - Reference clause numbers

2. **Ask Follow-Ups**
   - "Can you explain that further?"
   - "What if I can't meet this obligation?"

3. **Use Quick Questions**
   - Tap suggestions for common queries
   - Saves typing time

### For Finding Lawyers

1. **Be Precise with Location**
   - Use specific addresses for accurate distance
   - Consider traffic/travel time

2. **Check Multiple Profiles**
   - Compare fees and experience
   - Read client reviews
   - Check specializations match your need

3. **Contact Multiple Lawyers**
   - Many offer free consultations
   - Compare advice before deciding

### For Managing History

1. **Use Descriptive Titles**
   - Edit titles to be memorable
   - Include document type and date

2. **Regular Cleanup**
   - Delete old analyses you no longer need
   - Keeps history organized

3. **Export Important Analyses**
   - Download as PDF for records
   - Share with advisors or lawyers

## Troubleshooting

### Analysis Issues

#### "Analysis failed. Please try again."
**Causes**:
- Temporary API issues
- Network connectivity problems
- Document too large

**Solutions**:
- Check internet connection
- Try again in a few seconds
- For large documents, try smaller sections

#### "Missing API Key"
**Cause**: Gemini API key not configured

**Solution**:
- Go to Settings
- Enter your Gemini API key
- Or contact support for app key

#### PDF Upload Failed
**Causes**:
- Corrupt PDF file
- Encrypted/password-protected PDF
- Unsupported PDF version

**Solutions**:
- Try a different PDF
- Remove password protection
- Convert to plain text manually

#### No Text Extracted from PDF
**Causes**:
- Scanned PDF with poor quality
- Image-only PDF
- OCR failed

**Solutions**:
- Try higher quality scan
- Manually type text
- Use text version if available

### Login/Account Issues

#### Can't Sign In
**Solutions**:
- Check email and password
- Try "Forgot Password"
- Try Google Sign-In instead
- Clear browser cache

#### Analyses Not Syncing
**Causes**:
- Not signed in
- Network issues
- Firestore connection problem

**Solutions**:
- Ensure you're signed in
- Check internet connection
- Try manual sync (refresh button in history)

### Display Issues

#### Dark Mode Not Working
**Solution**:
- Clear browser cache
- Check Settings > Theme
- Try manual toggle in sidebar

#### Layout Looks Broken
**Solutions**:
- Refresh page (F5 or Cmd+R)
- Clear browser cache
- Try different browser
- Update browser to latest version

#### Visualizations Not Showing
**Solutions**:
- Wait for loading to complete
- Check "Visuals" tab/page
- Refresh page
- Try different simplification level

### Performance Issues

#### App Running Slow
**Solutions**:
- Close other tabs/apps
- Clear browser cache
- Delete old analyses
- Use Chrome or Firefox for best performance

#### Large Document Takes Forever
**Explanation**: Normal for very long documents (50+ pages)

**Solutions**:
- Be patient (can take 1-2 minutes)
- Consider analyzing sections separately
- Simplification level affects speed
